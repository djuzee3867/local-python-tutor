## Installation
1. **clone:**
   ```bash
   git clone https://github.com/djuzee3867/local-python-tutor
   cd local-python-tutor
   ```
2. install bottle
   ```bash
   pip install bottle
   ```
## run

   1. เข้าไปที่โฟลเดอร์ `main` ซึ่งเป็นโฟลเดอร์หลักที่ใช้ทำงาน
   ```bash
   cd main
   ```

   ```bash
   python bottle_server.py
   ```
  
   ```text
   http:// <---IP---\> :5000/visualize.html
   http://localhost:5000/visualize.html
   ```
